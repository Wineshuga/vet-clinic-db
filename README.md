# 📖 Vet Clinic DB <a name="about-project"></a>

**Vet Clinic DB** is a relational database created as an initial data structure for a vet clinic.In this project, you will find files containing commands used to create a table to store animals' information, insert some data into it, and query it.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

 - [Install PostgreSQL](https://www.postgresql.org/download)
 - psql command line

### Setup

Clone this repository to your desired folder:

```
  cd my-folder
  git clone https://github.com/wineshuga/vet-clinic-db.git

```

### Usage

To run commands in each file in psql command line.
